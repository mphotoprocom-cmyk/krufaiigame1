# ผจญภัยอักษร 3 หมู่

สื่อเกมวิชาภาษาไทย เรื่องอักษร 3 หมู่ (ไตรยางศ์) สำหรับนักเรียนชั้นประถมศึกษาปีที่ 2

## ความสามารถ

- คำถามสุ่มจากพยัญชนะไทย 44 ตัว จำนวน 12 ข้อต่อรอบ
- ตอบถูกได้ 1 คะแนน ตอบผิดลบ 1 คะแนน
- แสดงภาพกล้องเป็นพื้นหลังเต็มสนามเกม และเปิดกล้องอัตโนมัติเมื่อเริ่มเล่น
- ยกฝ่ามือและเลื่อนไปทางซ้าย กลาง หรือขวาเพื่อเลือกคำตอบ
- ค้างมือประมาณ 1 วินาทีเพื่อยืนยันคำตอบ
- มีปุ่มคลิก/แตะสำรองสำหรับอุปกรณ์ที่ไม่มีกล้อง
- เล่นได้บนคอมพิวเตอร์ แท็บเล็ต และโทรศัพท์
- ประมวลผลภาพกล้องบนอุปกรณ์ ไม่อัปโหลดภาพไปยังเซิร์ฟเวอร์

## วิธีเปิดเล่นในเครื่อง

กล้องทำงานเฉพาะเว็บไซต์ HTTPS หรือ `localhost` เท่านั้น จึงควรเปิดผ่านเว็บเซิร์ฟเวอร์:

```bash
python3 -m http.server 8000
```

จากนั้นเปิด `http://localhost:8000`

## GitHub Pages

โปรเจกต์นี้เป็น Static Site ไม่ต้อง build เมื่อ push ขึ้น GitHub แล้วให้ไปที่:

1. **Settings → Pages**
2. เลือก **Deploy from a branch**
3. เลือก branch `main` และโฟลเดอร์ `/ (root)`
4. กด Save

เว็บไซต์จะพร้อมใช้งานผ่าน HTTPS ซึ่งรองรับการขอสิทธิ์ใช้กล้อง

## เทคโนโลยี

- HTML, CSS และ JavaScript
- MediaPipe Hand Landmarker สำหรับตรวจจับฝ่ามือ
- ภาพฉากสร้างขึ้นเฉพาะสำหรับเกมนี้

การเปิดระบบตรวจจับมือครั้งแรกต้องเชื่อมต่ออินเทอร์เน็ตเพื่อโหลด MediaPipe และโมเดลตรวจจับมือ
